# CRUD PHP MySQL

A simple **CRUD (Create, Read, Update, Delete)** web app built with **PHP + MySQL**.

## Features
- Add new record
- View records
- Update record
- Delete record

## Tech Stack
PHP • MySQL • HTML • CSS

## Setup (Local)
1. Install XAMPP/WAMP
2. Copy project folder to:
   - XAMPP: `htdocs/`
3. Create a database (example): `productdb`
4. Update DB settings in `db.php` if needed:
   - server: `localhost`
   - username: `root`
   - password: `""`
   - dbname: `productdb`
5. Run in browser:
   `http://localhost/<project-folder-name>/`

## Notes
- Database credentials are set for local development (XAMPP default).
